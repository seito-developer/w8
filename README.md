## Q1.class
クラス Macbook を下記のように拡張してください。  
- プロパティcpuに任意の値（文字列”M1” または “M2” が設定できるようにしてください
- メソッド「deliveryTime」を追加してください。
  - このメソッドを実行すると、このMacbookが出荷するまでにかかる日数をほかのメソッドと同様HTML出力します。標準的な日数は10日間ですが、下記の条件が加わると追加で日数がかかります
  - cpuが”M2”のとき+1日
  - colorが”white” または “gold”のとき+1日
  - screenWidthが20以上のとき+2日
